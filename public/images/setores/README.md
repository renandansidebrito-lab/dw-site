# Imagens dos Setores

Esta pasta contém as imagens dos setores da empresa DW Granitos.

## Como adicionar suas imagens:

1. **Substitua as imagens placeholder** nesta pasta com suas imagens reais
2. **Formato recomendado**: JPG, PNG ou WebP
3. **Dimensões ideais**: 400x300 pixels (proporção 4:3)
4. **Nomes dos arquivos**:
   - `serraria-placeholder.jpg` → Substitua com imagem da serraria
   - `chapas-placeholder.jpg` → Substitua com imagem das chapas
   - `recortado-placeholder.jpg` → Substitua com imagem do recortado

## Onde encontrar as imagens:
- Fotos reais das instalações da empresa
- Imagens dos equipamentos em funcionamento
- Fotos dos produtos/serviços oferecidos

## Dicas:
- Use imagens de alta qualidade
- Mantenha a consistência visual entre as três imagens
- Certifique-se de que as imagens representem bem cada setor

Após substituir as imagens, elas aparecerão automaticamente na seção de setores do site.
