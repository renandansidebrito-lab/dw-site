## Como adicionar logos reais:

1. **Substitua os arquivos PNG placeholder** com os logos reais
2. **Formatos aceitos**: PNG (recomendado), JPG, WebP ou SVG
3. **Dimensões recomendadas**: 300x120 pixels
4. **Nomes dos arquivos**: Mantenha os mesmos nomes ou atualize no código

## Onde obter os logos:
- Site oficial das construtoras
- Materiais de marketing das empresas
- Contato direto com as construtoras

## Dicas importantes:
- Use logos com fundo transparente quando possível
- Mantenha a proporção original do logo
- Certifique-se de ter permissão para usar os logos
- Otimize o tamanho do arquivo para web

Após substituir os logos, eles aparecerão automaticamente na seção de parceiros comerciais do site.
